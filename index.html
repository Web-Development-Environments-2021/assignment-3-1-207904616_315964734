UserRegister Up

containing user info
username (optional)
String between 3 to 8 chars
example: omer123
firstname (optional)
String
example: gal
lastname (optional)
String
example: zamir
country (optional)
String
example: Israel
password (optional)
String between 5 to 10 chars, at least one of them is a number and a special char format: password
example: gal@123
email (optional)
String format: email
example: omer@gmail.com
image-url (optional)
String format: URL
example: https://facebook.com/profilepic/galzamir
